BinarizeByQuantile¶

class hybrid_learning.datasets.transforms.image_transforms.BinarizeByQuantile(quantile=None, num_pixels=None)[source]¶

Bases: hybrid_learning.datasets.transforms.image_transforms.ImageTransform

Set all but the given highest number of pixels / q-th quantile in an image to zero, rest to 1. Mind for RGB images: A pixel here means a pixel in one channel.

Init.

Parameters

  • quantile (float) – quantile of pixels to set to 1, rest is set to 0; overridden by num_pixels

  • num_pixels (int) – number of pixels with highest value to set to one, rest is set to 0
