Description

Standard and concept datasets derived from MS COCO dataset.

There are the following dataset handles, both derived from COCODataset:

  • KeypointsDataset: In: COCO images, GT: COCO annotations

  • ConceptDataset: In: COCO images, GT: masks for the given body parts

Masks are generated and saved in COCO style folder structure if they do not exist. For the documentation of the coco handle used under the hood, see the short COCO API documentation and the COCO dataset format doc.

Classes

BodyParts

Mapping of a visual concept description (body part) to COCO keypoint ID collections approximating it.

COCODataset

Attributes and functions common for keypoint based datasets derived from MS COCO.

ConceptDataset

Data handle for ground truth segmentations of visual concepts (body parts) generated from COCO keypoints.

KeypointsDataset

Handler for (a subset of) a COCO keypoints dataset.
