hybrid_learning.datasets.base.cross_validation_splits¶

hybrid_learning.datasets.base.cross_validation_splits(train_val_data, num_splits)[source]¶

Split dataset it into num_splits and collect tuples of (rest, split). This is useful for creating splits for cross-validation, where rest would be the training data, and split would be the validation data split.
