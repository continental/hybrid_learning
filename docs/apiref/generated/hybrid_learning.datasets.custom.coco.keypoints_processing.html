keypoints_processing

Description

Util functions to process, transform, and evaluate MS COCO keypoint information.

The main functionality is:

  • annotation transformation: pad_and_scale_annotation(), annotations_to_mask()

  • estimation of the size (in pixels) of a person in an image from an annotation: annotation_to_sizes(), annotation_to_tot_height()

  • some functions that may serve as filtering conditions: person_has_rel_size(), all_keypts_visible(), any_keypts_visible()

Attributes

COCO_STD_KEYPOINT_NAMES

The standard list of keypoint names for MS COCO.
