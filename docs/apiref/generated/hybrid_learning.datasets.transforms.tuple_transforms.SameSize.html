SameSize¶

class hybrid_learning.datasets.transforms.tuple_transforms.SameSize(interpolation='bilinear', resize_target=True)[source]¶

Bases: hybrid_learning.datasets.transforms.tuple_transforms.TupleTransforms

Given a tuple of input and target image, resize the target to the size of the input. Both input and target image must be given as torch.Tensor.

Init.

Parameters

    the interpolation method to use, parameter to torch.nn.Upsample;
