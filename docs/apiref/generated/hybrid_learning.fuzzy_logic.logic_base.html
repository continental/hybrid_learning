Description

Base classes for defining and operating on logics. This includes defining logical operations, defining a logic, and basic parsing of logical formulas.

Sub-modules

connectives

Base classes for some standard logical connectives with symbols and arity.

logic

Base classes for defining a logic.

merge_operation

Base classes and helper functions for defining logical operations.

parsing

Basic functionality for parsing string representations of logical formulas.
