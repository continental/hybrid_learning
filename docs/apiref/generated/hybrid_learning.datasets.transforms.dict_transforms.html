Description

Transformations on dict annotations containing scalars and same-sized masks.

The annotations that can be processed are supposed to be a dict with string keys and values of floats or numpy.ndarray (binary) masks.

The logical merging operations derived from Merge allow for concatenation. Using them, any operation involving intersection, union, and inversion of masks can be modelled. Scalar values in this case are treated as all-same-valued masks when mixed with mask tensors. For further information have a look at the Merge documentation.

Classes

AND

Intersection/AND operation on binary masks and scalars.

DictTransform

Basic transformation for dicts.

DropAnn

Drop the annotation with given key from the annotations dict.

FlattenDict

Return the value of the annotations dict at selected key.

Merge

Merge the binary masks / scalars of the input keys and store them under the specified output key.

NOT

Inversion/NOT operation on binary masks and scalars.

OR

Union/OR operation on binary masks and scalars.

RestrictDict

Restrict the annotation dictionary to the annotation items with featuring one of the selected keys.
