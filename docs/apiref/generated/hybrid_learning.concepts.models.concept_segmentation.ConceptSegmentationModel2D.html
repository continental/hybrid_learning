ConceptSegmentationModel2D¶

class hybrid_learning.concepts.models.concept_segmentation.ConceptSegmentationModel2D(concept, model, layer_id, in_channels=None)[source]¶

Bases: hybrid_learning.concepts.models.concept_detection.ConceptDetectionModel2D

A concept model that segments the concept in an image. This is solved by a localization model with window respectively kernel size \(1\times 1\).

For a usage example compare ConceptDetection2DTrainTestHandle.

Init.

Wrapper around init of a ConceptDetectionModel2D with fixed kernel_size of \(1\times1\). For details on the arguments see the init function of the super class ConceptDetectionModel2D.
