AbstractIoULoss¶

class hybrid_learning.concepts.kpis.AbstractIoULoss(reduction=<BatchReduction.mean: ('Reduce by mean', <function BatchReduction.<lambda>>)>, target_thresh=0.0)[source]¶

Bases: hybrid_learning.concepts.kpis.AbstractIoULike

Shared settings for intersection over union based losses. The difference to IoU based metrics is that only the targets are binarized, not the outputs. Thus, the function on the DNN outputs stays smoothly differentiable.

Init.

Parameters

  • target_thresh (float) – threshold to binarize targets

  • reduction (Union[hybrid_learning.concepts.kpis.BatchReduction, Callable[torch.Tensor, torch.Tensor]]) – reduction method to aggregate the instance-wise results of the batch; must be a callable on a tensor which reduces the 0th dimension; for examples see BatchReduction instances
