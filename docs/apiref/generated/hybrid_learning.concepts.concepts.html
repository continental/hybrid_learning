Description

Concept base class, concept types and corresponding sub-classes.

A concept is represented by dataset splits. Different concept types are implemented using sub-classes. Depending on the type, the data sets must fulfill different criteria. E.g. a segmentation concept requires segmentation masks as ground truth.

Concept

Representation of a concept with data and meta information.

Types the data of a concept can have.
