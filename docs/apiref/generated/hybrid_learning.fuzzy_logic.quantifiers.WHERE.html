WHERE

class hybrid_learning.fuzzy_logic.quantifiers.WHERE(tens_in_key, cond_in_key, dim=None, **kwargs)[source]

Bases: TorchOperation

Filter the dimensions in tensor a by the boolean values given in cond. If a is to be filtered in dimension d, the size of b must be 1 in all dimensions except for d, where it has the same dimensionality as a (may be e.g. the output of an ANY operation). If cond has dimensionality 1 in all dimensions, either the mask itself (cond==True) or a zero-shaped tensor (cond==True).
